# 001.两数之和

## 问题描述
给定一个整数数组 `nums` 和一个整数目标值 `target`，请你在该数组中找出 **和为目标值** `target` 的那 **两个** 整数，并返回它们的数组下标。

你可以假设每种输入只会对应一个答案。但是，数组中同一个元素在答案里不能重复出现。

你可以按任意顺序返回答案。

**示例 1：**
```
输入：nums = [2,7,11,15], target = 9
输出：[0,1]
解释：因为 nums[0] + nums[1] == 9 ，返回 [0, 1] 。
```

**示例 2：**
```
输入：nums = [3,2,4], target = 6
输出：[1,2]
```

**示例 3：**
```
输入：nums = [3,3], target = 6
输出：[0,1]
```

## 解题思路

### 方法一：暴力枚举
最直接的方法是使用两层嵌套循环，遍历数组中的每一个元素，寻找是否存在另一个元素与其和等于目标值。

时间复杂度：O(n²)，其中 n 是数组的长度。
空间复杂度：O(1)。

### 方法二：哈希表
使用哈希表可以将查找时间从 O(n) 降低到 O(1)。我们可以遍历数组，对于每个元素 `nums[i]`，检查哈希表中是否存在 `target - nums[i]`。如果存在，则返回它们的下标；否则，将当前元素及其下标存入哈希表。

时间复杂度：O(n)，其中 n 是数组的长度。
空间复杂度：O(n)，主要是哈希表的开销。
